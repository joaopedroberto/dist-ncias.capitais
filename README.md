### Descrição 

O projeto consiste na composição de uma tabela que conta com capitais 
brasileiras e a distância entre essas capitais. As capitais constante nesse projeto
são:

* São Luis 
* São Paulo
* Teresina
* Vitória 
* Porto Velho 
* Recife
* Rio Branco
